# PDF 字段标注工具

## 实现功能

- **加载 PDF**：选择本地 PDF，多页逐页渲染显示
- **画框**：在任意页拖拽画出矩形，满足最小尺寸后新增字段
- **编辑字段**：右侧列表点选字段，在表单中填写/修改 name、type、description
- **调整框大小**：选中字段后框变黄并显示四角手柄，拖拽手柄改大小
- **删除字段**：列表项右侧 × 或选中后按 Delete/Backspace
- **导出 JSON**：按 mission 格式导出 `fields.json`（name, type, description, x, y, width, height, page）

## 简短测试流程

1. **启动**：在项目根目录执行 `npx serve .` 或 `python3 -m http.server 8080`，浏览器打开对应地址
2. **加载**：点击「选择 PDF」选一个 PDF，确认多页正常显示
3. **画框**：在某一页空白处拖拽画矩形，确认出现蓝色框且右侧列表多一项、表单自动弹出
4. **编辑**：在表单填写 name/type/description，确认列表文案更新
5. **选中与调整**：点列表另一项，确认对应框变黄并出现四角手柄；拖拽某角改变大小，松手后框保持新尺寸
6. **删除**：点某列表项右侧 ×，确认该项与 PDF 上该框消失；或选中一项后按 Backspace，效果相同
7. **导出**：点击「导出 JSON」，确认下载 `fields.json` 且内容包含当前所有字段及坐标
